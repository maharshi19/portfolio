Machine Learning-Driven Cryptocurrency Trading System

Developed an advanced trading system in collaboration with an industry partner, leveraging machine learning models to optimize investment strategies and enhance decision-making in a volatile market environment. As a Data and ML Engineer, I collaborated with the industry partner to understand business requirements and tailor the system to real-world financial scenarios. I designed and implemented a robust data pipeline for real-time processing and analysis, integrating XGBoost and Random Forest models to predict market trends and optimize trading strategies. Achieved 82% accuracy with XGBoost and 80% accuracy with Random Forest through rigorous backtesting and hyperparameter tuning. Additionally, I utilized cloud-based data storage and visualization tools for real-time performance monitoring and decision support. This project enhanced my skills in industry collaboration, data engineering, machine learning (XGBoost, Random Forest), cloud computing, and financial data analysis.
